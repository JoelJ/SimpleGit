RefSpecs to be fetched. Blank for the default refSpec to be used. Each refSpec should be on its own line.

For example, if you want to checkout a GitHub pull request branch, you would use:

+refs/pull/*:refs/remotes/origin/pr/*

However, if you want to fetch pull request branch and compare it to a non- pull request branch, you would use:

+refs/heads/*:refs/remotes/origin/*
+refs/pull/*:refs/remotes/origin/pr/*
